**Save work files inside the existing project folder, never to Desktop.**

**Why:** Desktop fills up with orphaned files. Vas's roles each have a dedicated folder under `~/Documents/{role}/` and code projects live under `~/projects/{project}/`. Global CLAUDE.md: "Save files in existing project folders, not Desktop."

**How to apply:**
- MMA work → `~/Documents/mma-global/...`
- Moveo work → `~/Documents/moveo/...`
- Marketing Embeddings → `~/Documents/marketing-embeddings/...`
- Mbriyo → `~/Documents/mbriyo/...`
- Personal projects → `~/Documents/personal/...` or the specific `~/projects/` subdir
- If the right folder doesn't exist for the work, ask before creating; don't fall back to Desktop
