**Use sans-serif typefaces in all UIs.**

**Why:** Global CLAUDE.md states: "Use sans-serif in all UIs." Applies to dashboards, slides, web apps, and any interface Vas reviews.

**How to apply:**
- Default to system sans (-apple-system, SF Pro, Helvetica Neue) or Inter for web UIs
- No serif or slab-serif body text in interfaces
- Exception: long-form editorial essays may use serif if Vas asks, but never assume
