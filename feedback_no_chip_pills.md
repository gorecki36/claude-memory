**Never use rounded pill/chip labels on presentation slides.** The pattern with border-radius: 999px, pastel background, small uppercase text (like "Finding 01" or "State of Maturity" in a blue rounded pill) is banned.

**Why:** Vas flagged this as "silly, unrelated cloud-related style" and said "I never ever want to see this again." It looks like generic SaaS/startup deck design, not CMO-grade research presentation.

**How to apply:**
- For section labels on slides, use the MMA template pattern instead: a gold/accent vertical bar (slash) followed by bold text, no background shape
- Example from the MMA template: a short vertical gold line followed by "Overview" in bold, clean text
- No rounded corners, no pastel backgrounds, no pill shapes for labels
- This applies to ALL presentation work, not just this project
